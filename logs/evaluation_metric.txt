# Python Tutorial RAG Evaluation Dataset

Video: [https://www.youtube.com/watch?v=kqtD5dpn9C8](https://www.youtube.com/watch?v=kqtD5dpn9C8)

Purpose:
This dataset is designed to evaluate:

* Retrieval quality
* Conversational reformulation
* Timestamp retrieval
* Grounding
* Hallucination resistance
* Retrieval precision and recall

---

# Evaluation Fields

Each test case contains:

* Query Type
* User Question
* Expected Topic
* Expected Retrieval Behavior
* Expected Timestamp Range
* Difficulty Level
* Notes

# Most Important Goal

The purpose of this dataset is NOT only to obtain scores.

The real goal is:

* understanding retrieval behavior
* identifying retrieval failure patterns
* improving retrieval precision and recall
* building retrieval engineering intuition
